# 京东商品数据爬取工具

基于 DrissionPage 浏览器自动化的京东商品数据采集与 MySQL 存储工具。

## 功能

- **单独爬取** — 输入单个关键词，爬取京东搜索结果中按销量排序的商品数据
- **批量爬取** — 从 Excel 文件读取关键词列表，批量爬取
- **数据上传** — 将爬取的 Excel 数据上传到 MySQL 数据库

## 采集字段

| 字段 | 说明 |
|------|------|
| 商品文本 | 商品标题/描述 |
| 价格 | 售价 |
| 售出 | 销量 |
| 好评率 | 商品好评率 |
| 店铺名 | 商家名称 |
| 最低价 | 同款商品最低价 |
| 商品链接 | 商品详情页 URL |
| 商品图片 | 商品主图（Base64） |
| 搜索日期 | 数据采集日期 |

## 项目文件

| 文件 | 说明 |
|------|------|
| `122(1) - 副本.py` | 主程序，爬虫 + 交互菜单 |
| `mysql_uploader222222.py` | Excel → MySQL 上传模块 |

## 依赖安装

```bash
pip install requests DrissionPage beautifulsoup4 pandas openpyxl mysql-connector-python
```

## 使用方式

```bash
python "122(1) - 副本.py"
```

启动后显示交互菜单：

```
请选择爬取模式:
  1. 单独爬取模式      # 输入关键词，单个搜索
  2. 批量爬取模式      # 从Excel读取关键词，批量爬取
  3. 上传Excel到MySQL  # 将爬取结果上传数据库
  4. 退出程序
```

### 单独爬取

输入关键词后自动打开浏览器搜索京东，按销量排序后提取商品信息，结果保存为 Excel。

### 批量爬取

准备关键词 Excel 文件（每行一个关键词），程序读取后批量爬取，输出为 `批量结果-MM-DD-HH-MM.xlsx`。

### MySQL 上传

将爬取的 Excel 数据上传到 MySQL 数据库，每个 Excel 文件名作为独立的表名。

## 输出目录

爬取结果默认保存在 `C:\Users\<用户名>\Desktop\狗东数据`，程序会自动创建该目录。

## 技术要点

- 使用 DrissionPage 控制 Chromium 浏览器，模拟真实用户搜索
- 支持多线程并发处理（最大16线程）
- 商品图片以 Base64 编码存入 Excel / MySQL
- 数据表根据 Excel 文件名动态创建，支持中文表名
- 价格/销量字段同时保存原始值和数值，便于排序分析

## 注意事项

1. 需要本地安装 Chrome/Chromium 浏览器
2. 爬取速度受网络和京东反爬策略影响，建议合理设置线程数
3. 批量爬取时自动在每次搜索间添加延时，避免触发风控
4. 图片下载超时时间默认 5 秒
